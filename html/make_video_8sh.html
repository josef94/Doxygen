Detailed Description

The script to start a new video.

This script is used to start a new video. When this script is called from the handler, it takes an actual timestamp and start the video. The name of this new video involves the actual timestamp which is necessary to know, which video was started first and to calculate the timestamp when a vehicle driver is recognised.

A public variable which holds the actual date and time.

This variable is used to start a new video with the actual date and time. This is necessary to start every video with another name and to calculate other things with it.
